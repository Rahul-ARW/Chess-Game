# Two Player Chess - JavaScript Project

This is a two-player chess game built using:
- HTML
- CSS
- Vanilla JavaScript

## Features
- Two-player chess on one laptop
- Legal move validation
- Check detection
- Checkmate detection
- Stalemate detection
- Castling
- En passant
- Pawn promotion
- Move history
- Undo move
- Reset game

## Run locally
1. Download or copy all project files.
2. Open the folder in VS Code.
3. Open `index.html` in your browser.

## Files
- `index.html` → page structure
- `styles.css` → page styling
- `script.js` → full chess logic

## Optional VS Code extension
Install **Live Server** in VS Code if you want easier previewing.
